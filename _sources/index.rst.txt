Welcome to pycobra's documentation!
===================================

pycobra is a python package which implements predictor aggregation algoirthms. It also has a helpful suite of diagnostic and visualisation methods to analyse the aggregates and also the estimators used to create them. You can find the source-code on `GitHub <https://github.com/bhargavvader/pycobra>`_.

For examples and usage, the Jupyter notebooks illustrate the capacity of pycobra.

- `Basic Usage <https://github.com/bhargavvader/pycobra/blob/master/notebooks/pycobra.ipynb>`_
- `Advanced Usage <https://github.com/bhargavvader/pycobra/blob/master/notebooks/regression.ipynb>`_
- `Visualisation <https://github.com/bhargavvader/pycobra/blob/master/notebooks/visualise.ipynb>`_
- `Voronoi Tesselations <https://github.com/bhargavvader/pycobra/blob/master/notebooks/voronoi_clustering.ipynb>`_
